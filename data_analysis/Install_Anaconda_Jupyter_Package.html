1. How to download and install Anaconda and Jupyter Notebook?

Step 1: Download Anaconda

Navigate to the Anaconda website to download and install the app at https://www.anaconda.com/download. If you are a Mac user, please download the appropriate version (Intel or M1/M2).

Anaconda is an open-source package and environment management system for Python programming language. It is designed to simplify the process of installing, managing, and working with data science libraries, packages, and environments.

Please follow the video if you have issues downloading and installing Anaconda at https://www.youtube.com/watch?v=WUeBzT43JyY.

Step 2: Install and Launch Jupyter Notebook on Anaconda

Jupyter Notebook is an open-source web application that allows the creation and sharing of documents that contain live code, equations, visualizations, and narrative text. Its applications include data cleaning and transformation, numerical simulation, statistical modeling, data visualization, machine learning, and much more.

We will use Jupyter Notebook to write Python code for web data collection.

Step 3: Create a new Python "ipynb" file on Jupyter Notebook

Click on "New" and "Python 3(ipykernel)" under a folder where you can locate to write and run Python code.

Step 4: Write Python code on the file

Click on "+" to add a cell, write a simple code by typing "print("hello")", and click on "Run" to run the code line.

2. How to install Python packages or libraries for use in Jupyter notebook with Anaconda?

Python a library is just a collection of modules. A module is a single file that contains Python code, meant to be imported and used by other Python code. For example, the Python library "Pandas" provides high-performance, easy-to-use data structures like the DataFrame, and data analysis tools.

Step 1: Launch Anaconda Navigator and click on the "Environments" tab located on the left side of the window.

Step 2: Select the specific Python environment where you want to install the library. For instance, let's assume we are using the "python3916" environment. If you have just installed Anaconda, you might only see one Python environment named "base (root)".

Step 3: To search for a library, make sure the "All" option is selected in the drop-down menu next to the search bar. Enter the name of the library you want to install, for example, "bs4" for the BeautifulSoup library used for web scraping.

Step 4: The search results will display the library you searched for. To install the library, check the box next to its name.

Step 5: Click on the "Apply" button located at the bottom right corner of the Anaconda Navigator window.

Anaconda will initiate the installation process, and the selected library will be downloaded and installed into your chosen Python environment.

2.1. Alternative: Using Terminal to install Python libraries on Anaconda for Jupyter Notebook

Step1: Open Terminal

Step2: Type "conda env list" to check your Anaconda environments.

Step3: Type "conda activate" + your environment name to Activate the environment where you want to install the package.

Step4: Type "pip install" + the package name you want to install.

Now, your package has been successfully installed.
